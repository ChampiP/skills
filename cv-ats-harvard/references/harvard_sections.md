# Secciones del formato Harvard (orden y redacción)

Basado en la plantilla oficial "Harvard Bullet Point" (Harvard FAS — Mignone Center for Career Success), adaptada a reglas ATS.

## 1. Encabezado
Primera línea del documento (no header/footer de Word):
```
Nombre Apellido
Ciudad, País • correo@ejemplo.com • +51 999 999 999 • linkedin.com/in/usuario
```
Sin foto, sin dirección exacta, sin DNI ni estado civil.

## 2. Perfil profesional (opcional pero recomendado)
3-4 líneas. Especialidad + años de experiencia + 2-3 competencias clave alineadas a la vacante. Usar terminología exacta de la oferta cuando exista.

## 3. Educación
Orden cronológico inverso.
```
Institución                              Ciudad, País
Título / Grado, [GPA opcional]           Fecha de graduación
```
Incluir cursos relevantes u honores solo si aportan a la vacante.

## 4. Experiencia laboral
Orden cronológico inverso. Por cada puesto:
```
Empresa                                  Ciudad, País
Cargo                                    Mes Año – Mes Año
• Viñeta: verbo de acción + qué hiciste + resultado cuantificable
• Viñeta: ...
```
Verbos de acción sugeridos: Implementé, Desarrollé, Lideré, Reduje, Optimicé, Diseñé, Coordiné, Supervisé, Elaboré, Automaticé.

Evitar frases genéricas de funciones ("Encargado de..."); preferir logros ("Reduje el tiempo de conciliación documental de X a Y mediante...").

## 5. Proyectos relevantes (recomendado en perfiles técnicos)
```
Nombre del proyecto — descripción breve de una línea
• Viñeta de logro/impacto técnico
```

## 6. Habilidades
Agrupar por categoría, evitando listas sueltas sin contexto:
```
Backend y desarrollo: Python, JavaScript, TypeScript, React, PostgreSQL...
IA y automatización: RAG, LangChain, n8n, OCR...
Idiomas: Inglés (intermedio), Quechua (nativo)
```
Escribir las herramientas exactamente como aparecen en la oferta de trabajo objetivo.

## 7. Certificaciones
Si el CV original trae certificados como imágenes (diplomas, badges), convertir cada uno en una línea de texto:
```
Institución — Nombre del curso — Fecha
```
Ejemplo (a partir de una imagen de certificado Oracle Academy con fecha 29 nov 2024):
```
Oracle Academy — Artificial Intelligence with Machine Learning in Java — nov. 2024
```

## 8. Referencias (opcional)
Solo incluir si el usuario lo pide explícitamente o la vacante las solicita. Formato:
```
Nombre — Cargo, Empresa
Teléfono • correo
```
No repetir la sección ni duplicar referencias que ya aparecieron antes en el documento de origen.

## Notas de deduplicación
Los CV convertidos desde PDF a veces duplican secciones completas (ej. "Certificaciones" o "Referencias" aparecen dos veces con el mismo contenido). Antes de generar el documento final, consolidar cada sección en una sola aparición, conservando la versión más completa si hay diferencias menores entre las dos copias.
